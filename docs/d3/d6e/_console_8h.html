◆ criticalf()

template<typename ... Types>
void Zoe::criticalf ( const std::string &  format,
const Types &...  values 
)
inline

Prints a formatted critical message. The formatting can be escaped with \\. Every unescaped {index} is replaced by the element in values at index. If index is empty an intern index is used.
Examples:

criticalf("Foo {}", "bar"); //Output: "Foo bar"
criticalf("Foo {} {}", "bar", "boo"); //Output: "Foo bar boo"
criticalf("Foo {1} {0}", "bar", "boo"); //Output: "Foo boo bar"
criticalf("Foo {} {0} far", "bar", "boo"); //Output: "Foo bar bar far"
