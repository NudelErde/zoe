◆ bind()

void Zoe::OpenGLVertexBufferImpl::bind ( )
virtual

Binds this VertexBuffer if not already bound. The VertexBuffer is not directly bound to the OpenGLContext, but to the bound VertexArray. If no VertexArray is bound then the VertexBuffer is bound to the default VertexArray. If a VertexArray is bound to which this VertexBuffer is bound then it is also bound.
