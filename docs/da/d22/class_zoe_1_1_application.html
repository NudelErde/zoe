Detailed Description

Main Class
For the entry point, the developer needs to provide an implementation of the application class via Zoe::createApplication.
Only one instance can exist in the program.

Constructor & Destructor Documentation

◆ Application()

Zoe::Application::Application ( bool  withWindow = true)
explicit

Creates the Application and initializes the library.
withWindow can be set to false to start zoe without a graphics context. This should be used in tests.
Parts that use the graphics context will not load and will likely crash the program if they are used.
