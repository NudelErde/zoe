const std::set< std::string > &  options,
GraphicsContext *  context

Creates shader program by parsing the .glsl file.
The glsl file must contain vertex and fragment shader.
The preprocessor directive #shader is used to classify whether the following code is used for the vertex shader or fragment shader.

#shader [ vertex | fragment ]
Parameters
filepaththe .glsl file
optionsthe options used to create the shader
contextthe GraphicsContext. Must be OpenGLContext
