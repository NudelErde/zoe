Detailed Description

Implementation of IndexBufferImpl
OpenGl should be used if no better implementation is available.
An IndexBuffer stores the indices of the vertices used for drawing.
If the index buffer has the indices 0,1,2,2,3,0, a vertex buffer with 4 vertices can be used to draw a rectangle.

◆ bind()

void Zoe::OpenGLIndexBufferImpl::bind ( )
virtual

Binds this IndexBuffer if not already bound. The IndexBuffer is not directly bound to the OpenGLContext, but to the bound VertexArray. If no VertexArray is bound then the IndexBuffer is bound to the default VertexArray. If a VertexArray is bound to which this IndexBuffer is bound then it is also bound.
