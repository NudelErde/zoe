Detailed Description

The Scheduler executes Tasks and functions in the main thread. The main thread should call Scheduler::execute after adding at least one task or starting another thread.

◆ addTask() [1/2]

void Zoe::Scheduler::addTask ( Zoe::Task &&  task)
static

Adds a Task. The task will be executed until it is finished or it executes co_yield false;. The Scheduler is a cooperative Scheduler so the task has to call co_yield true; for other tasks to execute. If the task has an infinite while loop, it should call co_yield true; at least once per iteration.

Adds a function. The function will be executed once. The Scheduler is a cooperative Scheduler so the function should not take to long since it will block other tasks from being executed.

◆ execute()

void Zoe::Scheduler::execute ( )
static

Executes the tasks in this thread. This function returns after exit is called. Only one thread should call execute. If another thread executes this function, it prints an error and returns with no further effect.
