Detailed Description

The Scheduler executes Tasks and functions in the main thread. The main thread should call Scheduler::execute after adding at least one task or starting another thread.

◆ execute()

void Zoe::Scheduler::execute ( )
static

Executes the tasks in this thread. This function returns after exit is called. Only one thread should call execute. If another thread executes this function, it prints an error and returns with no further effect.
